Layer 1
Genesis Core
Algorithmic Concept Synthesis
▼
prt.py
Pure Relational Topology — E-Nodes & Relational Distinctions as a weighted directed asymmetric hypergraph. Builds simplicial complexes from abstract concepts.
invariants.py
Computes Betti numbers, cohomology classes, and persistence entropy using GUDHI/Ripser. Concepts are treated as cohomology classes in a multidimensional simplicial complex.
rse.py
Recursive Synthesis Engine — Semantic Hypergraph + Dynamic Embedding Manifold. Proposes conceptual blends via categorical pushouts and iterative synthesis.
manifold.py
Dynamic Embedding Manifold using geoopt (Poincaré / product-of-spheres). Provides hyperbolic/Riemannian embeddings for hierarchical conceptual structures.
aian.py
Anti-Imitation Adversarial Network — novelty-penalizing GAN. Discriminator checks similarity against frozen human-corpus encoder (sentence-transformers) to ensure non-human concept purity.
search.py
Divergent Beam Search — standard beam search with a structural-distance penalty added to the score, driving exploration away from known human concepts.
merit.py
Merit heuristics: MDL score, topological consistency, Schmidhuber compression progress (Δcompression), fractal dimension, edge-of-chaos ratio, and AIAN novelty score.
icvp/
Intersubjective Concept Valuation Protocol — Tendermint-style BFT consensus requiring >66% supermajority. Contains node.py (PoU sandbox), consensus.py, ledger.py (append-only), slashing.py (human-taint detector).

Layer 2
TMFT — Topological Magneto-Fluidic Translation
Hopfion synthesis in chiral ferromagnetic nematic colloid
▼
topology_to_field.py
Maps (crossing#, bridge_index) → (p, q) torus-knot params → target Hopf map on S³→R³. The conceptual-to-physical translation step.
coil_field.py
Biot–Savart + FEM solver for the 12-channel dodecahedral coil array. Computes spatial B-field as a function of 12 channel currents (nested tri-axial Helmholtz + Maxwell + RF micro-coils).
llg_solver.py
Landau–Lifshitz–Gilbert solver (MuMax3 wrapper) for magnetization dynamics in the Fe₃O₄ ferromagnetic nematic colloid.
q_tensor.py
Nematic Landau–de Gennes Q-tensor PDE solver (FEniCSx). Handles liquid-crystal director field coupled with magnetization for stable Hopfion formation.
fluid.py
Incompressible Navier–Stokes with active stress from the director field. Verifies Hopf texture stability within viscosity window (80–150 mPa·s) without turbulence.
photonics.py
FDTD / transfer-matrix model of the magneto-photonic lattice. Maps arousal/valence scalars to emission wavelength in 420–720 nm, 1–60 Hz pulse.
chamber.py
50 mm sapphire chamber boundary conditions. Enforces physical geometry constraints on all four solvers.
controller.py
Inverts target Hopfion → optimal coil currents via adjoint methods or differentiable physics (JAX-MD / Warp). Receives RetargetRequests from Layer 3 PID.
twin.py
Digital twin orchestrator — couples and sequences the four solvers (LLG, Q-tensor, fluid, photonic) for a coherent multiphysics simulation run.
↓
PhysicalRealization
↑
RetargetRequest
PhysicalRealization (L2→L3): achieved & target Hopf charge, charge error %, emission spectrum trajectory (420–720nm @60Hz), renderable mesh + per-vertex field tensors, mean B-field (mT). Physics boundary — L3 cannot reach back into solvers.

RetargetRequest (L3→L2, lateral): PID requests adjusted frequency if CLI overshoots. Clamped to 15–40 Hz resonant stability regime.
Layer 3
CRIM — Cognitive Receptivity & Integration Model
VR + biometrics → closed-loop Flow State targeting
▼
render.py
Hopf field → VR scene. Unity/Unreal bridge or Open3D (headless). Renders mesh + per-vertex field tensors from PhysicalRealization into an immersive visual experience.
prime_map.py
Maps Hopfion vectors onto 65 Universal Semantic Primes (Natural Semantic Metalanguage) plus embodied metaphor table. The semantic grounding layer.
phen_space.py
Multi-Dimensional Phenomenological Vector Space. Projects physical field data into a phenomenological representation space for cognitive integration scoring.
biometrics_lsl.py
Lab Streaming Layer ingest (real or simulated). Handles pupillometry, HRV, and GSR streams with sub-100ms sync across all biometric channels.
synthetic_subject.py
Simulated human subject: McSharry ECG model for HRV, Bach leaky-integrator GSR model, pupillary light-and-cognition model. Enables headless closed-loop testing.
cli.py
Cognitive Load Index estimator. Fuses biometric streams into a scalar CLI ∈ [0,1]. Target setpoint: 0.6 (Flow State).
pid.py
PID controller: CLI error → Immersion Intensity Score adjustment. May issue RetargetRequests to Layer 2 if frequency correction is needed. Drives difficulty/vignetting/narrative abstraction.
airlock.py
Abstraction Calibration Airlock. Gradual exposure gating — controls the rate at which novel non-human concepts are introduced to prevent cognitive overload.
metrics.py
Computes BHI (Biometric Homeostasis Index) and SDR (Semantic Distance Reduction) with confidence interval. These feed back to Layer 1 as Proof-of-Utility evidence.
