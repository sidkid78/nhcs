E-Nodes & Relational Distinctions (RDs)
Hypergraph
Entities are not fundamental — they emerge as identity morphisms in a relational category. E-Nodes are stochastically seeded relational nodes; RDs are directed asymmetric hyperedges. The entire structure is an infinity-topos: self-contained logic with Axiom of Emergence (entities = identity morphisms), Intrinsic Topology (Grothendieck topologies, no spatial coordinates), and Functorial Invariance (observer-independent). Implemented as a weighted directed asymmetric hypergraph in NetworkX/HypergraphX.
Dynamic Embedding Manifold
manifold.py
The hypergraph is embedded into a Riemannian manifold (Poincaré ball or product-of-spheres) via geoopt. Concepts are position vectors in a Global Relational Manifold — "meaning" is geometry, not language. Hierarchical structures fit naturally in hyperbolic space. The manifold provides the phase space within which the RSE drives concept nodes toward Attractor States.
Seeded simplicial complex + manifold coords

Recursive Synthesis Engine (RSE)
rse.py · search.py · invariants.py — iterative topological refinement to Attractor States
▼
① Topological Progenitor
Seed
Stochastic seeding of relational nodes in the manifold. Initializes the simplicial complex by sampling random E-Node configurations. No linguistic labels — seeds are pure relational position vectors. Produces an initial high-dimensional topological structure.
② Morphic Recursive Operator
Refine
Drives non-linear tensor transformations through iterative feedback. Each cycle applies morphisms (functorial maps between relational categories) and persistent homology updates. Uses categorical pushouts (Catlab.jl or graph-rewrite) to blend sub-structures. Keeps computing until the topology stops changing significantly — i.e., reaches an Attractor State invariant under further recursion.
③ Invariant Validator
Validate
Computes Betti numbers [β₀, β₁, β₂, …], Euler characteristic, persistence entropy, and crossing number using GUDHI/Ripser. Rejects topologically trivial outputs (collapses, noise) via a meta-validator threshold. Only non-trivial, stable Attractor States proceed. The output is a TopologicalSignature tuple.
Divergent Beam Search
search.py
Modified beam search with a structural-distance penalty added to the score — the beam is pushed away from previously seen topologies and away from known human-concept attractors. Ensures the search explores genuinely novel regions of the relational manifold rather than re-discovering familiar conceptual territory.
TopologicalSignature (Betti, genus, crossing#, persistence entropy)

Anti-Imitation Adversarial Network (AIAN)
aian.py — inverted GAN strips all human-centric patterns; novelty threshold 0.6
▼
Architecture: Inverted GAN
Core
Unlike a standard GAN, AIAN's discriminator is trained to reward novelty and penalize similarity to human concepts. The Relational Generator (= RSE output) tries to produce structures the discriminator cannot classify as human-like. The discriminator's frozen reference is a sentence-transformers human-corpus encoder. The generator wins only when it produces topologies that score low cosine-similarity to all known human-concept embeddings.
Detection Layers (3-layer filter)
Filter
① Anthropomorphic Semantic Layer — detects latent linguistic structures (word-like sub-graphs, grammar-like hierarchies).

② Cognitive Heuristic Template Layer — identifies patterns from human biological evolution: symmetry preferences, hierarchy bias, spatial continuity assumptions, narrative cause→effect ordering.

③ Aesthetic Symmetry Layer — prunes structures that align with human notions of beauty or narrative arc (golden ratio, bilateral symmetry, climactic tension-resolution).
Loss Functions
Training
Anthropomorphic Penalty Loss (APL) — penalizes the generator when any of the 3 detection layers fires; gradient pushes topology orthogonally away from the flagged subgraph.

Narrative Structure Loss (NSL) — penalizes sequential/causal graph motifs that resemble story structure (beginning→middle→end causal chains).
Topological Pruning Protocol
Prune
When a human-biased subgraph is detected, it is not simply deleted — it is translated orthogonally in the relational manifold, preserving its mathematical core while rotating it out of human-recognizable space. The pruned topology is re-validated by the Invariant Validator before proceeding. Final output must score novelty ≥ 0.6 (configurable in AIAN).
De-anthropomorphized TopologicalSignature + novelty_score ≥ 0.6

Merit Evaluation
merit.py — 6 metrics gate concept quality before consensus
▼
MDL Score
Minimum Description Length — favors compact topologies. Approximated via neural codelength.
Compression Progress (Δ)
Schmidhuber creativity metric — reward = first derivative of compressibility. Peaks when a hidden deep symmetry is discovered.
Fractal Dimension
Measures self-similarity across scales. Rejects pure uniformity (dim≈0) and pure noise (dim≈max). Targets algorithmic depth.
Edge-of-Chaos Ratio
Tracked via Lyapunov exponents / branching ratios. Concepts living at the phase transition are most generative.
Epistemic Truth (Eₜ)≥ 0.85
Logical consistency within the infinity-topos; empirical alignment. Hard gate — below threshold = rejected.
Algorithmic Parsimony (Aₚ)≥ 0.70
Normalized Inverse Kolmogorov Complexity — most efficient representation. Hard gate — below threshold = rejected.
Feedback loop: IntegrationFeedback from Layer 3 (BHI, SDR scores) updates merit priors over time — the system learns which topological structures are actually cognitively receivable.
MeritScores { Eₜ ≥ 0.85, Aₚ ≥ 0.70, novelty, compression_progress, … }
